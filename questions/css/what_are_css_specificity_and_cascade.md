# Что такое специфичность селекторов и каскад в CSS?

---

**Каскад** — это механизм, по которому браузер выбирает, какие CSS-правила применить к элементу, если на него влияет несколько правил одновременно.

**Специфичность** — это "вес" селектора. Чем выше специфичность, тем больше приоритет у правила.

---

#### Пример

```css
p {
  color: black;
}

.text {
  color: blue;
}

#main-text {
  color: red;
}
```

Если элемент имеет `id="main-text"` и `class="text"`, цвет будет `red`, потому что `id` имеет более высокую специфичность.

---

#### Приоритеты

- Inline-стили имеют самый высокий приоритет.
- `id` сильнее класса.
- Класс, атрибут и псевдокласс сильнее тега.
- Если специфичность одинаковая, побеждает правило, которое написано ниже.

---

#### Итог

Каскад отвечает за выбор итогового стиля, а специфичность помогает понять, какое правило победит. На собеседовании часто проверяют, умеет ли кандидат предсказывать итоговый стиль элемента.
