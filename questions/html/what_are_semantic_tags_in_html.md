# Что такое семантические теги в HTML?

---

**Семантические теги** — это HTML-теги, которые описывают смысл содержимого, а не только его внешний вид.

Например, `header`, `nav`, `main`, `section`, `article`, `aside` и `footer` помогают браузеру, поисковым системам и assistive technologies понять структуру страницы.

---

#### Пример

```html
<header>
  <nav>...</nav>
</header>

<main>
  <article>
    <h1>Заголовок статьи</h1>
    <p>Основной текст статьи.</p>
  </article>
</main>

<footer>...</footer>
```

---

#### Зачем они нужны?

- Улучшают читаемость HTML-кода.
- Помогают SEO, потому что структура страницы становится понятнее.
- Улучшают доступность для screen reader.
- Делают страницу проще для поддержки.

---

#### Итог

Семантическая разметка делает HTML не просто набором блоков, а понятной структурой документа. На собеседовании важно сказать, что семантика влияет не на внешний вид напрямую, а на смысл и доступность страницы.
